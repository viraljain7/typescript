> # Typescript Notes
>
> convert typescript file into javascript

```js
tsc filename.ts
```

> Typescript

```js
const anExampleVariable: string = "Hello World";
console.log(anExampleVariable);
```

> Javascript file

```js
"use strict";
const anExampleVariable = "Hello World";
console.log(anExampleVariable);
```
